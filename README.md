# 🌦️ Premium Smart City Weather Dashboard

Welcome to the **Premium Smart City Weather Dashboard**. This application provides a high-end, visual experience for checking real-time weather and 5-day forecasts for any city globally.

### 🌍 Live Demo
[**View Live Project**](https://smart-city-weather-dashboard-eta.vercel.app)

---

## ✨ Features
- **Premium Glassmorphism UI**: A modern, translucent interface with dynamic background themes.
- **Search by City**: Get real-time weather data with a single search.
- **Geolocation Detection**: Automatically detects your city on load.
- **Interactive Forecast**: Filter by conditions (Rainy, Cloudy, Hot, Cold) and sort by temperature or day.
- **Responsive Design**: Optimized for everything from mobile phones to high-resolution desktops.

---

## 🛠️ Technical Highlights
- **Tech Stack**: HTML5, CSS3 (Glassmorphism), Vanilla JavaScript.
- **Data Source**: OpenWeatherMap API integrated directly.
- **Array Methods**: Heavy use of `map()`, `filter()`, and `sort()` for efficient data rendering.
- **Dynamic Theming**: Responsive background gradients that shift based on current temperature.

---

## 🚀 Getting Started

### Run Locally
Since this is a pure static application, getting started is instant:
1. **Clone the repository:**
   ```bash
   git clone https://github.com/arjavj1105/Smart-City-Weather-Dashboard.git
   ```
2. **Open the Dashboard**:
   - Simply double-click `index.html` to open it in your browser.
   - The API key is already integrated, so you can start searching immediately!

### Deployment
This project is configured for **Zero-Config Deployment** on Vercel or Netlify. Since the API key is integrated into `config.js`, no environment variables or build scripts are required.
